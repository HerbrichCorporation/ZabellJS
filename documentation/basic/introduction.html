Introduction Einleitung About this Framework Über dieses Framework Simplicity is a framework for building surfaces. Special attention was paid to that no other libraries are used in order to keep the lines of code as small as possible. That promises performance and readability. It is also advised to use vanilla Javascript. Simplicity ist ein Framework um Oberflächen zu bauen. Dabei ist extra darauf geachtet worden das keine weiteren Bibliotheken verwendet werden, um die Lines of Code möglichst gering zu halten. Das verspricht Performance und Lesbarkeit. Es wird auch geraten vanilla Javascript zu nutzen. Build management was deliberately avoided because the ES6 modules are used. Each view is lazy loaded and the other attached components as well. Thus the framework achieves a excellent page load latency. Auf Bildmanagement wurde bewusst verzichtet, da die ES6 Modules verwendet werden. Jede View wird lazy geladen und die weiteren angehängten Komponenten ebenfalls. Somit erreicht das Framework eine ausgezeichnete Latenz beim Laden von Seiten. The architecture has been kept particularly close to the web component and HTML specification. This framework can therefore be seen as an extension of the Web Component specification. Die Architektur ist extra besonders nahe an der Web-Component und HTML Spezifikation gehalten worden. Somit kann dieses Framework als eine Erweiterung der Web-Component Spezifikation gesehen werden.
