Introduction Einleitung About this Framework Über dieses Framework This Framework is used to be without NPM or other libraries, because of performance issues. It is kept simple to be as most performant as it can be. It has no compiler, all is used natively direct from the Browser. This way it is as performant as the Browser can be. Dieses Framework wird aufgrund von Leistungsproblemen ohne NPM oder andere Bibliotheken verwendet. Es ist einfach gehalten, um so leistungsfähig wie möglich zu sein. Es hat keinen Compiler, alles wird nativ direkt verwendet aus dem Browser. Auf diese Weise ist es so performant wie der Browser nur sein kann.
